# Платформа подбора вакансий

Веб-сервис для поиска работы и подбора персонала с интеграцией крупнейших работодателей.


## Возможности проекта
* **Умный фильтр:** Быстрый подбор вакансий через систему опроса.
* **Каталог предложений:** Наглядные карточки с зарплатой и графиком работы.
* **Партнеры:** Готовые интеграции с компаниями Яндекс Еда, Самокат и Пятерочка.
* **Адаптивность:** Сайт корректно отображается на ПК и мобильных устройствах.


## Стек технологий
* **HTML5** — разметка и структура.
* **Tailwind** — стилизация и сетки.
